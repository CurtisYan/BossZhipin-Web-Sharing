# BOSS 职位长图分享浏览器扩展

把 BOSS 直聘网页端正在浏览的职位生成类似 App 分享页的长图，并尽量自动抓取“微信扫码分享”悬浮弹层里的二维码。

## 安装

1. 打开 Chrome 或 Edge 的扩展管理页。
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. 打开“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择这个文件夹：`boss-job-share-extension`。

## 使用

1. 打开 `https://www.zhipin.com/web/geek/jobs`。
2. 在左侧点开一个职位，让右侧出现职位详情。
3. 点击浏览器工具栏里的扩展图标，可以选择“复制到剪贴板”或“下载 PNG”。
4. 点击页面右下角的“做长图”。
5. 如果选择复制，成功后图片会进入剪贴板，可以直接粘贴到微信、聊天窗口或文档里。如果浏览器阻止写入剪贴板，会自动下载 PNG 兜底。

## 关于二维码

扩展会自动尝试触发“微信扫码分享”的悬浮弹层并抓取二维码。如果网页结构变化导致没有抓到，可以先把鼠标放到“微信扫码分享”上，让二维码弹出来，再点击“生成并复制长图”。

## 如果看不到悬浮按钮

扩展刚加载后，已经打开的 BOSS 页面可能还没有注入脚本。刷新 BOSS 页面一次即可。工具栏弹窗按钮也会在需要时自动补注入脚本。

## 当前范围

- 支持 BOSS 直聘网页端职位详情页和推荐页右侧职位详情区域。
- 不依赖固定 class，采用多策略识别职位名、薪资、城市、经验、学历、职位描述、公司和地址。
- 生成的是本地 PNG，不会上传你的职位信息。
